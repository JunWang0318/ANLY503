Sub-Question 4: Is there a relationship between the age-adjusted cancer rate and income, smoking, diabetes, and obesity?


We first computed the correlation matrix and show it on a heatmap as below. From the visualization, we can see that the top two features that are related to the age-adjusted cancer rate are smoking and diabetes.



Figure 21 (Seaborn)


Then we decided to conduct hypothesis tests on the relationship between smoking and age-adjusted cancer rate as well as between diabetes and age-adjusted cancer rate.


First, for the smoking data, each point represent the single record for one state in a specific year. The H0 hypothesis is that there is no relationship between smoking and cancer rate, while H1 is there is a relationship between smoking and cancer rate.


We built the linear regression for the model and the following are the chart and statistic information about the regression.



Figure 22 (ggplot)


The results indicate that all the coefficients for this model are significant, however, the R-squared is about 0.1018 which means there are only 10% variation of the data points can be explained by this linear model.


Similarly, for the Diabetes data, each point represent the single record for one state in a specific year. The H0 hypothesis is that there is no relationship between diabetes and cancer rate, while H1 is there is a relationship between diabetes and cancer rate.


We also built the linear regression for the model.



Figure 23 (ggplot)


Same to the smoking case, all the coefficients are significant, but the R-squared is about 0.02796 which means there are only 3% variation of the data points can be explained by this linear model. Therefore, based on our two regression models, there is no linear relationship between the age-adjusted cancer rate and smoking or diabetes.

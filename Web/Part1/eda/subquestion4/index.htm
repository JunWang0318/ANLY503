Sub-Question 4: Is There a Relationship between the Cancer Incidence Rate and Income, Smoking, Diabetes, and Obesity?


We first created an interactive correlation matrix and showed it in a heatmap below. The viewer is able to select features that he or she is interested in then the corresponding correlation matrix will be generated. From a comprehensive comparison, we selected all the features and can see that the top two factors related to the cancer rate are smoking and diabetes.

Then we decided to conduct hypothesis tests on the relationship between smoking and cancer incidence rate as well as between diabetes and cancer rincidence ate by using a linear regression. As a result from part 1 , even though all parameters are statistically significant, the R-squared for both regression lines were really small. Therefore, we were not able to conclude that there is some relationship between smoking and cancer incidence rate, or between diabetes and cancer incidence rate.


In addition to the previous linear regression, we decided to plot the multiple regression for the cancer incidence rate by using plotly 3D scatter plot.

The results indicate that all the coefficients for this model are significant, however, the R-squared is about 0.1018 which means there are only 10% variation of the data points can be explained by this linear model.


Similarly, for the Diabetes data, each point represent the single record for one state in a specific year. The H0 hypothesis is that there is no relationship between diabetes and cancer incidence rate, while H1 is there is a relationship between diabetes and cancer incidence rate.


We also built the linear regression for the model.



Figure 23 (ggplot)


Same to the smoking case, all the coefficients are significant, but the R-squared is about 0.02796 which means there are only 3% variation of the data points can be explained by this linear model. Therefore, based on our two regression models, there is no linear relationship between the age-adjusted cancer rate and smoking or diabetes.

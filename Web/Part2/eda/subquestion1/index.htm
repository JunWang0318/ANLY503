Sub-Question 1: What is the overall trending for cancer incidence rate, mortality rate, and death rate?


We first plot the combined line chart for the cancer count and cancer rate.



Figure 4 (Plotly)


In the chart above, the blue line represents the number of mortalities due to cancer and orange line represents the number of incidences. The green line is the trending for mortality rate and the red line is the trending for incidence rate, both with the unit of count per 100,000 people.



Figure 5 (Plotly)


The above 3D-scatter plot is the compact version of the line charts, with the size and color of the dot represents the cancer rate. The larger size and darker color dot have a large value of the cancer rate. The viewer can select the rate using the dropdown filter on the upper left corner. For both charts, we can see that even though the total number of counts for both incidence and mortality are increasing since 1999, the actual rates are decreasing.


To obtain a clear overall trending, we introduce a new variable feature - death rate which is computed by the number of mortalities divides the number of incidences for each year.


Figure 6 (Matplotlib/Pandas)


From the line chart above, we can see that there is a significant drop for the death rate since 1999, then there a fluctuation since 2009 and stabilized after 2013.
